# Nayepankh Foundation - Frontend Web Application 🕊️

A modern, responsive Single Page Application (SPA) developed as a portfolio project for the Web Developer position at the **Nayepankh Foundation**. 

This project demonstrates advanced frontend architecture without relying on heavy frameworks like React or Vue. It is built entirely with vanilla web technologies to ensure maximum performance, accessibility, and clean code structure.

## ✨ Key Features

* **Custom SPA Router:** Seamlessly switches between 6 different pages (Home, About, Features, Campaigns, Join Us, Contact) without reloading the browser.
* **Scroll Animations:** Utilizes the `IntersectionObserver` API to trigger smooth fade-in animations only when elements enter the viewport, optimizing performance.
* **Interactive Live Chat UI:** A custom-built floating chat widget with simulated bot responses and automatic scroll-to-bottom logic.
* **Real-time Form Validation:** The "Join Us" form features live Regex-based email validation and visual UI feedback (success/error states) before submission.
* **Dynamic Number Counters:** JavaScript-driven counter animations that run automatically when the user navigates to the Home view.
* **Fully Responsive Design:** Adapts flawlessly to mobile, tablet, and desktop screens using modern CSS Grid and Flexbox layouts.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Advanced layouts (Grid/Flexbox), custom properties (variables), and keyframe animations.
* **Vanilla JavaScript (ES6+):** DOM manipulation, state management, event handling, and custom routing.
* **Deployment:** Vercel

## 🧠 Technical Highlights for Recruiters

1. **Zero Dependencies:** Built without jQuery, Bootstrap, or heavy JS frameworks to showcase a deep understanding of core web fundamentals.
2. **State & DOM Management:** The `app` JavaScript object encapsulates all logic, preventing global scope pollution and keeping the code modular and easy to maintain.
3. **Modern CSS:** Heavy reliance on CSS variables (`:root`) makes global theme changes (like updating brand colors) instant and scalable.
